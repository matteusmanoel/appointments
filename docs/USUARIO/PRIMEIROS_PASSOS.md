# Primeiros passos — Plano Essencial

Se você está no plano **Essencial**, não tem acesso ao WhatsApp com IA. O NavalhIA oferece **painel + link público de agendamento**: o cliente agenda 24h pelo link e você gerencia tudo no painel.

Siga estes passos para deixar sua barbearia pronta para receber agendamentos online.

---

## 1. Serviços

Cadastre os serviços que você oferece: nome, preço e duração.

- Acesse **Serviços** no menu.
- Clique em **Novo serviço** (ou equivalente).
- Preencha: nome (ex.: Corte, Barba), preço (R$) e duração em minutos.
- Salve. Repita para cada serviço.

**Checklist:** [ ] Todos os serviços que quero oferecer estão cadastrados.

---

## 2. Barbeiros

Cadastre os barbeiros que atendem na sua unidade.

- Acesse **Barbeiros** no menu.
- Adicione cada barbeiro com nome e status (ativo / pausa / inativo).
- O cliente vai escolher o barbeiro na hora de agendar.

**Checklist:** [ ] Barbeiros cadastrados.

---

## 3. Horários

Defina o horário de funcionamento para o sistema saber em quais dias e horários há vagas.

- Acesse **Configurações** (ou o local onde está **Horários** / **Funcionamento**).
- Configure, por dia da semana, se abre ou fecha e o horário de início e fim.
- Salve.

Se você já configurou horários na tela de **Integrações** (porque em algum momento teve ou testou Pro/Premium), essa mesma configuração é usada para o link de agendamento.

**Checklist:** [ ] Horário semanal salvo.

---

## 4. Link público de agendamento

O cliente acessa um link fixo da sua barbearia para escolher serviço, barbeiro e horário.

- Em **Configurações**, procure por **Link de agendamento**, **Página pública** ou **URL do agendamento**.
- Copie o link (ex.: `https://app.navalhia.com.br/b/minha-barbearia`).
- Use esse link no Instagram, site, cartão, etc.

**Checklist:** [ ] Link copiado e divulgado onde quiser.

---

## Resumo Essencial

| O que fazer | Onde |
|-------------|------|
| Cadastrar serviços (nome, preço, duração) | Serviços |
| Cadastrar barbeiros | Barbeiros |
| Definir horário de funcionamento | Configurações / Horários |
| Pegar e divulgar o link de agendamento | Configurações |

Quando quiser **WhatsApp com IA**, lembretes e follow-up automático, faça upgrade para o plano **Profissional** e use o [Tutorial WhatsApp](WHATSAPP_SETUP.md).
